# Marketing Funnel & Conversion Performance Analysis

## 📌 Project Overview
This project focuses on analyzing marketing campaign data to understand customer conversion behavior and identify opportunities to improve marketing performance.

The analysis was performed using the Bank Marketing dataset and aims to answer important business questions such as:

- What is the overall conversion rate?
- Which customer segments convert the most?
- Which months generate higher conversions?
- How can marketing campaigns be optimized?

This project was completed as part of the **Data Science & Analytics Internship Program at Future Interns**.

---

# 🎯 Objectives

- Analyze customer conversion performance.
- Identify high-converting customer segments.
- Study conversion trends across different demographics.
- Evaluate campaign effectiveness.
- Provide actionable business recommendations.
